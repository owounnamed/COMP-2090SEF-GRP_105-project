# Unieats

This project is a **Food Review Application** designed to help users discover and review restaurants within the Kowloon district, specifically the areas around **Ho Man Tin** and **Mong Kok**. The application provides both **User** and **Admin** functionalities, where users can post reviews, upload images, and browse restaurant recommendations, while admins can manage restaurant information and reviews.

## Features

### **User Features:**

* **Post Reviews:** Users can submit food reviews for restaurants they have visited. Reviews can include ratings, textual feedback, and uploaded images.
* **Restaurant Recommendations:** The application suggests restaurants based on users' previous dining history. Similar restaurants are recommended, making it easy for users to explore new places based on their preferences.
* **Sorting & Filtering:** Users can sort restaurants by various criteria such as rating, price, or distance. Filters can be applied to narrow down the restaurant options based on factors like cuisine type and user budget.
* **Budget Selection:** Users can set their desired budget range, and the application will suggest restaurants within the selected price range.
* **Random Recommendation:** A fun feature allows users to receive a random restaurant suggestion based on their preferences.

### **Admin Features:**

* **Restaurant Management:** Admins have the ability to add, update, and delete restaurant listings. They can also manage restaurant-related reviews and ensure accurate information is provided.
* **Review Management:** Admins can modify or remove inappropriate reviews and ensure that the content posted is relevant and appropriate for the platform.

### **Additional Features:**

* **Map Integration:** A dynamic map displays restaurant locations. Users can view the restaurant's position relative to their current location or search by specific area.
* **User-Friendly Interface:** The application is built using **Python OOP** (Object-Oriented Programming) principles, ensuring a scalable and maintainable structure.

## Technologies Used

* **Python (OOP)**: The application is developed using Object-Oriented Programming principles for clean and maintainable code.
* **Map Integration**: The app integrates a map view for easy navigation of restaurants in the Kowloon area.
* **SQLite**: The app uses SQLite to store user reviews, restaurant information, and other relevant data.

## Areas Covered:

* **Ho Man Tin** and **Mong Kok**.
